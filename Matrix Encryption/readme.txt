This is an encryption method I created, I decided to call it "Matrix Encryption" or "Dimensional Encryption".
There's definitely other encryption methods that are much more efficient and secure, I just made this for fun in high school.

Explanation:
Say you want to encrypt the word "Dog".
When you give the word "Dog" to the program it creates a 5 dimensional array ([][][][][]), each array in the 5 dimensional array has a length of 10.
This 5 dimensional array gets filled with random letters, numbers, and symbols. A number is generated that points to different spots in the array that
contains your word, this string of numbers if the key.
The length of the key is always (5 x length of word), so in this case the key would be a string of 15 numbers, for example, it would looks something like this:
948273648293847
This key is a reference to different places in the 5 dimensional array.
So it would be like this:
94827 36482 93847
Which stands for
[9][4][8][2][7] [3][6][4][8][2] [9][3][8][4][7]
Assuming this is the correct key for this matrix and the encrypted word is dog, the result would be:
Element [9][4][8][2][7] = D
Element [3][6][4][8][2] = o
Element [9][3][8][4][7] = g
Then you get your original word.
This works because in order to make any sense of the array you need the exact key, otherwise the result will just be gibberish.
And even if you have the key, the key is useless unless you have the correct matrix, so you need both in order to get a result.
The program allows you to save and load matrixes so it's not just one-time use.
